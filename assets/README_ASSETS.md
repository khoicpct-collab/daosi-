## üõ†Ô∏è How to Add/Update Assets

### For Material Images:
1. Take photos of materials or find stock images
2. Resize to 300x300 pixels
3. Optimize for web (compress if needed)
4. Save as JPG with quality 80-90%

### For Icons:
1. Use SVG format for scalability
2. Keep consistent stroke width (2px)
3. Use currentColor for stroke/fill to inherit CSS colors
4. Optimize SVG with SVGO or similar tools

### For Templates:
1. Create in PowerPoint with FlowSim branding
2. Include placeholder slides for each slide type
3. Save as .pptx files
4. Test with the export functionality
